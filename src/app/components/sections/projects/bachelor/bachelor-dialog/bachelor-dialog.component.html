Bachelor Thesis:
Controlling virtual agents in an urban traffic simulation

For my bachelor thesis at FSU Jena, I used reinforcement learning (specifically, Proximal Policy Optimization) to teach virutal cars how to navigate a simulated road network.

My work included

  • Working with a customizable Unity library.
  • Researching and evaluating different learning strategies, including reinforcement learning and genetic algorithms.
  • Working within Unity's ML-Agents framework.
  • Building a learning agent within a specifically designed environment.
  • Training a reinforcement learning model and evaluating progress.
  • Understanding underlying mathematic models and algorithms.
  • Writing a scientific thesis explaining my process and visualising the resulting data.

Abstract Reinforcement learning has been an active area of research for more than a decade and continues to evolve. This bachelor thesis applies reinforcement learning to the problem of controlling virtual agents in an urban traffic simulation. It successfully implements learning agents using Proximal Policy Optimization, a specifically designed curriculum and different approaches to reward shaping in an existing road generation and simulation Unity library called Eltrilo. The thesis will explain the necessary theoretical background regarding neural networks and reinforcement learning, detail the implementation of learning agents and conduct experiments using multiple agent configurations as well as provide an outlook over possible modifications. Furthermore, an analysis of the systemâ€™s performance will be provided, concluding that while agents learn to navigate the environment well, reinforcement learning exhibits several key weaknesses that make it suboptimal for urban traffic simulation: long training times, high computational intensity and poor control over specific agent behaviour.
