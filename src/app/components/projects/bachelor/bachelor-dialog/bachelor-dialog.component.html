Bachelor Thesis:
Controlling virtual agents in an urban traffic simulation

Abstract

English German

Reinforcement learninghas been an active area of research for more than a decadeand continues to evolve. This bachelor thesis applies reinforcement learning to theproblem of controlling virtual agents in an urban traffic simulation. It successfullyimplements learning agents usingProximal Policy Optimization[31], a specificallydesigned curriculum and different approaches to reward shaping in an existing roadgeneration and simulation Unity library calledEltrilo. The thesis will explain thenecessary theoretical background regarding neural networks and reinforcement learn-ing, detail the implementation of learning agents and conduct experiments usingmultiple agent configurations as well as provide an outlook over possible modifi-cations. Furthermore, an analysis of the system’s performance will be provided,concluding that while agents learn to navigate the environment well, reinforcementlearning exhibits several key weaknesses that make it suboptimal for urban trafficsimulation: long training times, high computational intensity and poor control overspecific agent behaviour.
