Intershop PWA

My responsibilites while developing the Intershop PWA include:

  • Developing, maintaining and testing clean, reusable angular components.
  • Developing and testing services that interact with various REST APIs.
  • Developing a large NGRX store.
  • Using RXJS to write functional, observable-driven code.
  • Developing, debugging and using angular schematics.
  • Developing and maintaining end-to-end tests with cypress.
Formly Integration

The Problem:
We realized that our complex network of custom form components, including validation and styling, was becoming increasingly hard to maintain and customize - both for ourselves and for our clients. We needed a flexible, easily customizable way to write forms.
The Solution: I researched the Formly library which allows for programmatic definition of forms via typescript.
